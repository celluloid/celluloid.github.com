Overview

Wakes up sleepy threads so that they can check their mailbox Works like a ConditionVariable, except it's implemented as an IO object so that it can be multiplexed alongside other IO objects.

def signal
  @sender << PAYLOAD
  nil
rescue IOError, Errno::EPIPE, Errno::EBADF
  raise DeadWakerError, "waker is already dead"
end

def wait
    byte = @receiver.read(1)
    raise DeadWakerError, "can't wait on a dead waker" unless byte == PAYLOAD
rescue IOError, RuntimeError
    raise DeadWakerError, "can't wait on a dead waker"
end
